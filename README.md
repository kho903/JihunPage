# JihunPage

A personal introduction page built with React.

## Current Scope

- Build the frontend first
- Add a Spring Boot backend later
- Implement signup and session-based login as a future feature

## Tech Stack

- React
- Vite
- JavaScript
- Bootstrap
- ESLint

## Project Structure

```text
JihunPage/
├── frontend/
│   ├── public/
│   ├── src/
│   │   ├── assets/
│   │   ├── App.jsx
│   │   ├── index.css
│   │   └── main.jsx
│   ├── package.json
│   └── vite.config.js
└── README.md
```

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

The development server runs at:
```text
http://localhost:5173
```
